# percolation_cpp
Этот проект реализует моделирование явления перколяции в двумерных сетках с использованием алгоритма поиска в ширину (BFS) на языке C++.

В данной модели исследуется возникновение связных центров, образующих пути для заряда, в случайных сетках и определение перколяционного порога — вероятности, при которой появляется сквозной путь через всю систему.
